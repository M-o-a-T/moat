# References

Reference documentation for MoaT-MQTT console scripts and programming
API.

## Console scripts

- `moat_mqtt_pub` : MQTT client for publishing messages to a broker
- `moat_mqtt_sub` : MQTT client for subscribing to a topics and
  retrieved published messages
- `moat_mqtt_broker` : Autonomous MQTT broker

## Programming API

- `mqttclient` : MQTT client API reference
- `broker` : MQTT broker API reference
- `common` : Common API
